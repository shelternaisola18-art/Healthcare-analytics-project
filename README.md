# Healthcare Analytics Project

## Overview
Analysis of hospital data to identify revenue and operational improvements.

## Tools
Excel, SQL, Power BI

## Key Insights
- Revenue concentrated in few conditions
- High length of stay reduces capacity
- Demand varies by season
